Agar aplikasi bisa berjalan seperti Native App di iPhone Anda, ikuti langkah ini.

## Syarat Utama
Aplikasi harus sudah di-deploy (punya URL website sendiri, misal: `https://my-app.vercel.app`). Pastikan Anda sudah melakukan `vercel deploy --prod` di laptop.

## Langkah Install

1.  **Buka Safari**
    Buka **Safari** di iPhone Anda. (Wajib Safari, jangan Chrome/Firefox untuk instalasi).

2.  **Buka URL Aplikasi**
    Ketik alamat website aplikasi Anda yang didapat dari Vercel.

3.  **Tap Tombol Share**
    Lihat di menu bar bawah, tekan ikon **Share** (Kotak dengan panah ke atas).
    
    ![Share Icon](https://symbols.getvecta.com/stencil_80/43_share-ios-icon.6866a2fb71.svg)

4.  **Pilih "Add to Home Screen"**
    Scroll menu ke bawah sampai menemukan **"Add to Home Screen"** atau "Tambah ke Layar Utama".
    
    *(Ikonnya kotak tambah "+")*

5.  **Konfirmasi**
    Tap **Add** di pojok kanan atas.

## Catatan
Jika setelah install layar masih putih atau loading lama:
1.  Pastikan internet lancar saat pertama kali buka.
2.  Tutup aplikasi (swipe up) dan buka lagi.
